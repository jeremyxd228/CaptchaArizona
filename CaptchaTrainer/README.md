# CaptchaTrainer — Android x64

Тренажёр капчи для Android (arm64-v8a / x86_64), аналог xsoft.

## Функции
- Кнопка **N** — открыть капчу
- 5-значный случайный код, каждая цифра своего цвета
- Таймер 15 секунд с прогресс-баром
- Кнопки: N, Payday, Fast, Cam, Stop, Lag, Chat, Old Captcha
- Кнопки **Принять** / **Отмена**
- Лог в реальном времени (зелёный = верно, красный = ошибка)
- Статистика: процент верных, средний ввод, средний первый символ, лучшее время
- После ввода автоматически открывается следующая капча

---

## Способ 1 — GitHub Actions (самый простой, не нужен ПК)

1. Зарегистрируйся на https://github.com
2. Создай новый репозиторий (New repository)
3. Загрузи все файлы этого проекта в репозиторий
4. Перейди во вкладку **Actions**
5. Запусти workflow **Build APK**
6. После завершения скачай APK из **Artifacts**

---

## Способ 2 — Android Studio (на ПК)

1. Скачай Android Studio: https://developer.android.com/studio
2. Открой папку `CaptchaTrainer` как проект
3. Подожди пока синхронизируются Gradle зависимости
4. Нажми **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. APK будет в: `app/build/outputs/apk/debug/app-debug.apk`

---

## Установка APK на телефон

1. На телефоне включи **Установка из неизвестных источников**
   (Настройки → Безопасность → Неизвестные источники)
2. Перекинь APK на телефон
3. Открой файл и установи
